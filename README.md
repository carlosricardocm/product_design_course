# Basic Workshop on Smart Products — Course Materials

Teaching materials and 3D / fabrication files for the undergraduate course
**"Taller Básico de Productos Inteligentes" (Basic Workshop on Smart Products)**,
part of a Product Design degree.

The course teaches students to design and build interactive (smart) products by
combining form, basic electronics, microcontroller programming, simulation, and
digital fabrication, through three escalating projects: an interactive lamp, an
interactive flowerpot, and an interactive alebrije.

## Contents

- `models/` — 3D models and fabrication files (e.g. STL, STEP, laser-cut layouts)

## How to use

Browse the folder for the material you need. The `models/` folder contains files
ready for 3D printing or laser cutting; check each subfolder for the recommended
material and settings.
